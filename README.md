# ClassHelper
플래시 없이 웹을 이용하여 학급 운영에 도움되는 자리/청소(개발중) 바꾸기를 이용해 보세요!

## 설치 방법
/ClassHelper.sql 을 DB에 Import 하고 /application/config/database.sample.php 의 파일명을 database.php로 변경 후 DB 접속 정보를 올바르게 변경하세요.

## 기능
* 고정자리
* 자리 1:1 변경
* 엑셀로 학생 정보 입력
* 서버에 자리 데이터 저장
* 자리 화면 이미지 파일로 저장
